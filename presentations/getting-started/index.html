What does VAMPD do?

  • Sets up the Server
  • Deploys
  • Installs
  • Imports
  • Updates

Deploys

Places your code where it belongs

  • Creates a database
  • Pulls git repo at a revision
  • Templated: drush make (optional)

Installs

Installs your site

  • drush si ...
  • Custom templated settings.php with your settings
  • Allows for additional settings.php

Imports

Imports an existing database.

Updates

drush updb

Starter Site

Basic Drupal 7 Core Standard install pulling from Github


{
  "name": "example",
  "description": "The example role that gets added to every machine.",
  "chef_type": "role",
  "json_class": "Chef::Role",
  "default_attributes": {},
  "override_attributes": {
   "drupal": {
      "sites": {
        "example": {
          "active": true,
          "deploy": {
            "action": ["deploy", "install"]
          },
          "drupal": {
            "settings": {
              "docroot": ""
            }
          }
        }
      }
    }
  }
}

Exisiting Make Site With a DB to import


{
  "name": "lightning",
  "description": "This lightning role uses make and an install profile to install the lightning profile.",
  "chef_type": "role",
  "json_class": "Chef::Role",
  "default_attributes": {},
  "override_attributes": {
    "drupal": {
      "sites": {
        "lightning": {
          "active": true,
          "deploy": {
            "action": ["deploy", "import", "update"]
          },
          "drush_make": {
            "api": "2",
            "files": {
              "default": "build-lightning.make",
              "core": "drupal-org-core.make"
            },
            "template": true
          },
          "drupal": {
            "settings": {
              "profile": "lightning",
              "docroot": ""
            },
            "db_file": "/vagrant/sites/lightning/raw_db.sql"
          },
          "repository": {
            "host": "github.com",
            "uri": "git@github.com:timodwhit/lightning.git",
            "revision": "7.x-1.x"
          }
        }
      }
    }
  }
}

Using VAMPD?

First time

  1. Install vagrant, ChefDK, virtualbox and VAMPD
  2. Install vagrant plugins
  3. Copy and edit chef/roles/[site_name].json
  4. vagrant up
  5. edit /etc/hosts with your new site
  6. visit site_name.local in your browser
